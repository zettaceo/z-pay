Como funciona

Cobrar nunca foi tão simples

Três passos para receber do seu cliente — com checkout, API e eventos em tempo real.

🏪
1. Crie sua conta merchant
Onboarding rápido com KYB leve (manual no MVP), API Keys e configuração de webhooks.
🔗
2. Gere uma cobrança
Via dashboard ou API: valor, métodos aceitos e regras de split. Link instantâneo.
📡
3. Confirme e concilie
Webhooks primeiro, idempotência e log de eventos para conciliação e auditoria.
